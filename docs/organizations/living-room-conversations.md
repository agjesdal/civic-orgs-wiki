# Living Room Conversations

**Domain:** livingroomconversations.org  
**Categories:** [Bridging Political Divides](../categories/bridging-political-divides.md)  
**Date added:** 2026-05-14

## Mission

Free, open-source conversation guides that help small groups across political, generational, racial, and other divides have structured one-hour conversations using shared agreements.

## Notes

Lower-touch, peer-to-peer model — closer to a self-serve toolkit than facilitated workshops.
